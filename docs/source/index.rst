Cool Seq Tool
=============

Cool Seq Tool description

.. code-block:: pycon

   >>> import cool_seq_tool
   >>> print("example")
   "example"

Cool Seq Tool is a library created to support efforts to `normalize variation descriptions <https://github.com/cancervariants/variation-normalization/>`_ and `model gene fusions <https://github.com/cancervariants/fusion-curation/>`_. It is developed primarily by the `Wagner Lab <https://www.nationwidechildrens.org/specialties/institute-for-genomic-medicine/research-labs/wagner-lab>`_. Full source code is available on `GitHub <https://github.com/GenomicMedLab/cool-seq-tool>`_. TODO link to the live fusion instance?
